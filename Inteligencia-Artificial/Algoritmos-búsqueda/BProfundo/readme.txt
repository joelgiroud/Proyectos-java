El algoritmo de búsqueda en profundidad (DFS, Depth First Search) es un algoritmo que permite recorrer o buscar en estructuras de datos como árboles o grafos. 
El algoritmo DFS funciona de la siguiente manera: 

    Explora una rama del grafo o árbol lo más profundo posible.
    Retrocede para explorar ramas alternativas.
    Utiliza una pila para administrar los nodos que necesitan ser explorados.

El algoritmo DFS es útil para: Encontrar rutas, Encontrar ciclos, Encontrar componentes conectados en un grafo, Atravesar un árbol en un orden determinado,
Resolver acertijos como laberintos. 
La complejidad temporal del algoritmo DFS es O(V+E), donde (V) representa el número de vértices y (E) el número de aristas en el gráfico.
